Attendance Instructions
  1. Create a session for each class meeting using the form above.
  2. During class, navigate to the session and click the key icon to generate an OTP.
  3. You must be physically present in the classroom (connected to the correct Wi-Fi) to generate an OTP.
  4. Share the generated OTP with students, who must also be physically present.
  5. Students must enter the OTP within the time limit (15 minutes) to mark attendance.
  6. If class is cancelled, click the X icon to automatically mark all students present.
  7. Click the users icon to manually edit attendance statuses for each student.
OTPs expire after 15 minutes for security reasons.
