Key Features

Wi-Fi Location Validation

Mark attendance only when you're physically present in class using Wi-Fi router validation.

OTP Verification

Secure attendance marking with one-time passwords generated by instructors.

Real-time Analytics

Comprehensive reports and analytics to monitor attendance patterns and compliance.

Leave Request Management

Submit and track leave requests with automated approval workflows.

Attendance Alerts

Automatic notifications when attendance falls below required thresholds.

Academic Calendar Integration

System automatically adjusts for holidays, exams, and special academic events.

How It Works

For Students
  1. Attend your class and connect to the classroom Wi-Fi network.
  2. Access the attendance marking page, which only opens when you're in the correct location.
  3. Enter the OTP provided by your instructor to verify your presence.
  4. Track your attendance records and receive alerts if you're falling below requirements.
  5. Submit leave requests for planned absences with supporting documentation.
For Faculty
  1. Create and manage courses with classroom location and Wi-Fi router information.
  2. Generate OTPs for each class session while physically present in the classroom.
  3. View real-time attendance data and reports for all students.
  4. Approve or reject leave requests with notifications to students.
  5. Identify students at risk of attendance shortfalls and take proactive measures.
