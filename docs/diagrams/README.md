# System Diagrams

This folder contains important architecture and flow diagrams for the Attendance Management System.

## Files

- **architecture-diagram.png** - Complete system architecture showing all components and their interactions
- **sequence-diagram.png** - Attendance marking process flow diagram

## Adding Diagrams

To add the diagrams to this project:

1. Place `architecture-diagram.png` in this directory
2. Place `sequence-diagram.png` in this directory
3. Commit and push to GitHub

The README.md will automatically reference these diagrams once they are added to this folder.
